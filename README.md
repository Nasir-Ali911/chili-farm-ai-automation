# chili-farm-ai-automation
Automated chili export pipeline using a RAG-based agent. Leveraging n8n, Gemini AI, and Supabase, the system extracts data from unstructured PDFs to provide real-time shipment updates via WhatsApp. This solution reduces manual entry and empowers farmers with accurate, AI-driven logistics tracking.
